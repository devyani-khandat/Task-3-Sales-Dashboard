# 📊 Sales & Profit Analysis Dashboard (Task 3)

## 🔍 Objective
To perform deep-dive analysis and build an interactive dashboard using Power BI.

## 📁 Dataset
The dataset contains sales, profit, discount, category, and region data.

## 📊 KPIs
* Total Sales
* Total Profit
* Total Discount

## 📈 Analysis
* Sales by Category
* Profit by Category
* Sales by Region

## 🎯 Key Insights
* Technology category has the highest sales
* West region generates maximum revenue
* Profit varies significantly across categories

## 🛠 Tools Used
* Power BI

## 📌 Conclusion
This dashboard helps in understanding business performance and enables better decision-making through interactive filters.
